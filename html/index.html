  • worker - can monitor Traffic Lights in read-only mode. Can not send commands to Traffic Lights controller.
  • operator - can operate Traffic Lights in read-write mode. Can send most commands to Traffic Lights controller. But some commands are forbidden.
  • sysadmin - complete control over Traffic Lights controller.
